# Time extractor

Мини-программа для расчёта активного времени сотрудников в чатах 1С-Коннект.

## Что умеет

- Загружать JSON-файл с диалогами.
- Загружать данные из HTTP-сервиса 1С через локальный proxy.
- Определять сотрудников компании в выбранном периоде.
- Позволять выбрать нужных сотрудников для расчёта.
- Считать расчётное активное время по сообщениям.
- Показывать результат по сотрудникам.
- Открывать читабельные диалоги по каждому сотруднику.
- Экспортировать результат в CSV.

## Правила расчёта

Программа считает не фактическое время за компьютером, а расчётное активное время работы с обращениями.

Для каждого несистемного сообщения сотрудника программа:

1. Находит предыдущее несистемное сообщение в этом же диалоге.
2. Считает промежуток времени до ответа сотрудника.
3. Если интервал больше лимита, берёт максимальный активный интервал.
4. Если интервал меньше минимального порога, берёт минимум на сообщение.

Настройки по умолчанию:

- Максимальный активный интервал: 15 минут.
- Минимум на сообщение: 30 секунд.
- Системные сообщения не учитываются.
- Компания сотрудников: ООО Селекта Сервис.
